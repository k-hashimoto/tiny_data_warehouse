# サイドバーガイド

サイドバー（エクスプローラーパネル）は3つのセクションに分かれています。各セクションの機能を説明します。

---

## 1. テーブル（Tables）

メインの DuckDB データベース（`app.db`）のテーブルを、スキーマ別に表示します。

### ヘッダーボタン

| ボタン | 操作 |
|--------|------|
| **＋** | 新しいスキーマを作成 |
| **↑** | CSV ファイルをインポート |
| **↻** | テーブル一覧を更新 |

### スキーマの操作

- フォルダアイコンをクリックしてスキーマを展開・折りたたみできます。
- スキーマを右クリックするとコンテキストメニューが表示されます：
  - **Drop Schema (CASCADE)** — スキーマとその中のテーブルをすべて削除します。組み込みの `main` スキーマは削除できません。

### テーブルの操作

| 操作 | 結果 |
|------|------|
| テーブル名をクリック | カラム一覧を展開・折りたたみ |
| テーブル名をダブルクリック | `SELECT * FROM … LIMIT 100` を新しいタブで開く |
| ホバー → ℹ︎ アイコン | テーブルメタデータモーダルを開く |
| 右クリック → **SELECT * FROM …** | ダブルクリックと同じ |
| 右クリック → **Drop Table** | テーブルを削除（確認ダイアログあり） |

### カラム一覧

テーブル名の横のシェブロン（▶）をクリックすると、カラム名とデータ型が表示されます。

### CSV インポート（ドラッグ＆ドロップ）

`.csv` ファイルをテーブルセクションにドラッグ＆ドロップすると、CSV インポートダイアログが開きます。

---

## 2. dbt

[dbt](https://www.getdbt.com/) が `~/.tdwh/db/dbt.db` に生成したテーブルをスキーマ別に表示します。

### ヘッダーボタン

| ボタン | 操作 |
|--------|------|
| **🔗 / Unlink** | dbt データベースをアタッチ・デタッチします。アタッチ中は `FROM dbt.<スキーマ>.<テーブル>` でクエリできます |
| **↻** | dbt テーブル一覧を更新 |

### ステータスバッジ

- **attached**（緑）— dbt.db がアタッチされており、クエリ可能な状態です。
- **read-only**（青）— dbt.db は閲覧のみの状態で読み込まれています。
- **running**（黄色バナー）— `dbt run` が現在実行中です。

### 自動更新

`dbt run` が完了して `dbt.db` が更新されると、エクスプローラーが自動的に変更を検知してテーブル一覧をリフレッシュします。手動操作は不要です。

### dbt テーブルの操作

| 操作 | 結果 |
|------|------|
| テーブル名をダブルクリック | `SELECT * FROM dbt."スキーマ"."テーブル" LIMIT 100` を新しいタブで開く |
| ホバー → ℹ︎ アイコン | テーブルメタデータモーダルを開く（dbt テーブルは閲覧のみ） |
| ホバー → 🗑 アイコン（スキーマ） | スキーマ全体を削除 |
| ホバー → 🗑 アイコン（テーブル） | テーブルを削除 |

---

## 3. 保存済みクエリ（Saved Queries）

保存した SQL スクリプトをフォルダ別に表示します。

### ヘッダーボタン

| ボタン | 操作 |
|--------|------|
| **↻** | スクリプト一覧を更新 |

### スクリプトの操作

| 操作 | 結果 |
|------|------|
| スクリプト名をクリック | スクリプトを新しいタブで開く |
| ホバー → ▶ ボタン | スクリプトをすぐに実行 |
| ホバー → 💾 ボタン | 現在のエディタ内容でスクリプトを上書き保存 |
| 右クリック → **上書き保存** | 💾 ボタンと同じ |
| 右クリック → **名前を変更** | スクリプト名をインライン編集 |
| 右クリック → **Delete** | スクリプトを削除（確認ダイアログあり） |

### フォルダ

スクリプト名に `/` を含めて保存すると、自動的にフォルダが作成されます（例：`analysis/monthly_revenue`）。

### エディタからの保存

SQL エディタで **Ctrl/Cmd + S**（またはエディタツールバーの保存ボタン）を押すと、現在のクエリをスクリプトとして保存できます。名前の入力を求められます。`/` を区切り文字として使うとフォルダに保存できます。

---

## テーブルメタデータ

任意のテーブルの ℹ︎ アイコンをクリックすると、テーブルメタデータモーダルが開きます。

| 項目 | 編集 |
|------|------|
| テーブル名 | — |
| スキーマ | — |
| テーブルの説明 | ✓（app.db のテーブルのみ） |
| カラム名・データ型 | — |
| カラムのコメント | ✓（app.db のテーブルのみ） |

dbt テーブルのメタデータは閲覧のみです。
